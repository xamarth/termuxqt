# Termux Cutie

Cute Theme For Termux

### Installation

- Update termux packages and install `git`

```
pkg upgrade && pkg install git
```

- Clone this repository

```
git clone --depth=1 https://github.com/xamarth/termuxqt.git
cd termuxqt
chmod +x setup.sh
./setup.sh --install
```

### Uninstallation

```
./setup.sh --uninstall
```
